### Submodules  
Projekt zawiera w sobie inne repozytorium (googltest) i zwykły clone go nie pobiera.  
Można to zrobić na dwa sposoby:  
1. ```
git clone <url>
git submodule init
git submodule update
```
2. 
`git clone --recurse-submodules <url>`
